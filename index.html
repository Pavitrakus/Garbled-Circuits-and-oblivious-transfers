How Flipping Works

C = M ⊕ B. If you flip bit i in C, then upon decryption:
M' = C' ⊕ B = (M ⊕ B ⊕ Δ) ⊕ B = M ⊕ Δ

The attacker flips exactly the bits needed to change S→S, A→A, N→L etc. without ever touching the key B.

HOMOMORPHIC POWER

RSA Multiplication

RSA is multiplicatively homomorphic:
C₁ × C₂ = (M₁ᵉ)(M₂ᵉ) = (M₁×M₂)ᵉ mod n

Multiply ciphertexts → multiply messages. An attacker can double a bid from 13→26 by computing C × 2ᵉ mod n — all without the private key.
